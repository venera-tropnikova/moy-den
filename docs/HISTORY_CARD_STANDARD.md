# Историческая справка v1.0

## Назначение

Короткая историческая история дня, которую интересно читать утром за 15–20 секунд.

## Структура

### Фото-карточка

- заголовок раздела «Историческая справка»;
- фотография;
- название события;
- навигация.

### Текстовая карточка

- без заголовка «Историческая справка»;
- название события;
- дата в формате:
  `23 июля · 1962`
- короткая история;
- 💡 интересный факт;
- строка:
  `Источник ↗ …`

## Правила текста

- простой русский язык;
- без энциклопедического стиля;
- одно событие = одна история;
- один интересный факт;
- факт понятен без дополнительных объяснений;
- текст читается за 15–20 секунд.

## Правила дизайна

- премиальный минимализм;
- иерархия через типографику и воздух;
- без лишних декоративных элементов;
- открытая карточка без заголовка раздела;
- фото-карточка остаётся входом в раздел.

## UX

После прочтения человек должен запомнить хотя бы один интересный факт и захотеть открыть следующую карточку.
