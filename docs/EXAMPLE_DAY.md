## 4. Редакционная оценка подборки

Проверка не только баллов событий, но и **тройки целиком**:

| Критерий | Результат |
|----------|-----------|
| Разнообразие тем | технологии связи · космос/природа · российская наука |
| Нет повторов | не два «спутника/планеты»; комета ≠ Telstar |
| Разные эпохи | 1962 / 1995 / 1874 |
| Мир + национальное | WORLD + WORLD + RU |
| Три разные вещи | прямой эфир через океан · великая комета · электрический свет |

**Почему не Telstar + Kepler + комета:** повтор космической темы.  
**Почему Лодыгин вместо Kepler в тройке:** сильное национальное RU + другая эпоха и эмоция; Kepler — запасная альтернатива.

## 9. Как выглядит карточка

### Закрыто
- Eyebrow: `Историческая справка`
- Изображение главного события (Telstar)
- Вопрос:  
  `Что произошло`  
  `23 июля?`

### Открыто (главное событие)
- Дата: `23 июля · 1962`
- Изображение Telstar (компактнее)
- Summary (2–3 предложения)
- `Источник: NASA ↗`
- Далее (в будущей реализации концепции): остальные 2 события подборки

### Empty-state
Для этой даты **не** используется — есть verified-события.

## 10. Почему выбраны именно эти события

1. **Telstar** — максимальный утренний вау и желание рассказать: «мир впервые увидел друг друга в прямом эфире».  
2. **Хейла — Бопп** — другая эмоция (небо, природа/космос), сильнейшая картинка, не дублирует «спутник связи».  
3. **Лодыгин** — сильное национальное RU, XIX век, наука/изобретение; подборка не из одних мировых космических фактов.

Альтернатива на замену №2: **Kepler-452b (2015)** — если важнее «новое знание о планете», чем визуал кометы.
