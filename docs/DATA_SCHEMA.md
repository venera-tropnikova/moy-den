### history-events-ru.json

#### Структура

События в `eventsByDate["MM-DD"]` — массив объектов.

Поле `sources` — массив объектов источника:

```json
{
  "title": "NASA Science — C/1995 O1 (Hale-Bopp)",
  "shortTitle": "NASA Science",
  "url": "https://science.nasa.gov/solar-system/comets/c-1995-o1-hale-bopp/"
}
```

| Поле | Обязательность | Описание |
|------|----------------|----------|
| `title` | да | Полное название источника |
| `url` | да | Ссылка |
| `shortTitle` | нет | Короткая подпись для карточки (если `title` > 35 символов) |

Основной источник для UI — **`sources[0]`** (должен быть валидным: `title` + `url`).

#### Правила заполнения

##### Правила для sources

- Первый элемент массива (`sources[0]`) считается основным.
- Именно он отображается в карточке.
- Остальные источники используются для проверки и могут отображаться в будущем на отдельной странице или в расширенном режиме.
- В карточке: `Источник: … ↗` — текст из `shortTitle`, иначе из `title` (без URL на экране).
- Если `title.length > 35` — заполнять `shortTitle` (кратко, до 35 символов).
- Если `shortTitle` нет — в UI использовать `title`.
