# Мой день
Современное веб-приложение для ежедневной информации и планирования.
## Возможности

-  Погода
-  Важные даты
-  Историческая справка
-  Открытие дня
-  Анекдот дня
-  Поздравления
-  План на день

## Технологии

- HTML
- CSS
- JavaScript
